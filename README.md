# backend-ksl

## Run the project locally using node
Requiremets:
- [node 20+](https://nodejs.org/en/download)

Install depencies:
```shell
npm install
```

Generate prisma client:
```shell
npm run prisma-client
npm run seed-db # (optional) database seeding
```

Run the server in dev mode:
```shell
npm run dev
```
